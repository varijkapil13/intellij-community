Reports a setter of a property with a backing field that doesn't update the backing field.

The quick-fix adds assign to the backing field.

Example:


  class Test {
      var foo: Int = 1
          set(value) {
          }
  }

After the quick-fix is applied:


  class Test {
      var foo: Int = 1
          set(value) {
              field = value
          }
  }
