Reports hashMapOf function or HashMap constructor calls replaceable with the constructor of EnumMap.

Using a corresponding functions makes your code simpler.

The quick-fix replaces the function call with the constructor of EnumMap.

Example:


  enum class E {
      A, B
  }

  fun getMap(): Map<E, String> = hashMapOf()

After the quick-fix is applied:


  enum class E {
      A, B
  }

  fun getMap(): Map<E, String> = EnumMap(E::class.java)
