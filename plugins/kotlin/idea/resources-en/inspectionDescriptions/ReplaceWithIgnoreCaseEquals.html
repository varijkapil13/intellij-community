Reports case insensitive comparison that can be replaced with equals(..., ignoreCase = true).

By using equals() you won't have to allocate any extra strings with toLowerCase()/toUpperCase() to do the comparison.

The quick-fix replaces toString with string template.

Note: May change semantics for some locales.

Example:


  fun main() {
      val a = "KoTliN"
      val b = "KOTLIN"
      println(a.toLowerCase() == b.toLowerCase())
  }

After the quick-fix is applied:


  fun main() {
      val a = "KoTliN"
      val b = "KOTLIN"
      println(a.equals(b, ignoreCase = true))
  }
